\def\AbstractEN{
    This thesis introduces a custom Virtual File System (VFS) with versioning and encryption capabilities, implemented using the FUSE library adjusted for C++.
    The VFS enhances file management by adding password protection and encryption, enabling secure storage and retrieval of sensitive data.
    Builtin versioning on the other hand allows users to access and restore previous file versions effortlessly.
    The modularity of the VFS also allows for potential integration of additional features in the future, further improving its utility and adaptability.
    The VFS is efficient and thoroughly tested, providing a reliable solution across various platforms.
}

\def\AbstractCS{
    Tato bakalářská práce se věnuje vytvoření vlastního virtuálního souborového systému (VFS) s podporou verzování a šifrování a to prostřednictvím knihovny FUSE upravené pro použití s C++.
    Sifrování přidává možnost zamykat jednotlivé soubory a složky heslem pro bezpečné uchovávání citlivých dat.
    Verzování na druhou stranu umožní uživatelům snadno uchovávat a obnovovat předchozí verze souborů.
    Modularita VFS také umožňuje potenciální integraci dalších funkcí v budoucnu, čímž dále zlepšuje jeho užitečnost a využitelnost.
    Součástí bakalářeské práce je důkladný návrh, implementace a testování pro spolehlivé řešení napříč různými platformami.
}
