\def\AbstractEN{
    This thesis presents a custom Virtual File System built using the FUSE library, customized for use with C++.
    The VFS is modular in design, enabling easy extension with new features.
    Two prototype modules for versioning and encryption are also provided.
    With its password/key protection and encryption capabilities, the VFS improves file management, facilitating secure storage and retrieval of sensitive data.
    In addition, built-in versioning functionality allows users to access and restore previous file versions with ease.
    Thorough testing has demonstrated the VFS's efficiency and reliability on various platforms.
}

\def\AbstractCS{
    Tato bakalářská práce se věnuje vytváření virtuálního souborového systému prostřednictvím knihovny FUSE upravené pro použití s C++.
    Zmíněné VFS je moudlární a tedy umožňuje snadné přidání nových funkcí.
    Dále jsou poskytnuty dva prototypy modulů pro podporou verzování a šifrování.
    Sifrování přidává možnost zamykat jednotlivé soubory a složky heslem pro bezpečné uchovávání citlivých dat.
    Verzování na druhou stranu umožní uživatelům snadno uchovávat a obnovovat předchozí verze souborů.
    Součástí bakalářeské práce je také důkladný návrh, implementace a testování pro spolehlivé řešení napříč různými platformami.
